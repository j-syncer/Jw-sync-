How to merge your JW Library backups

  1. Export a backup. Create a backup in JW Library on each device (menu → Settings → Backup & Restore → Create Backup).
  2. Add your main backup. Open JW Sync and add the backup from your primary device as the first file.
  3. Add your other backups. Add the .jwlibrary files from your other devices as additional files.
  4. Review the merge. Check the impact summary and choose which version wins for any conflicting notes.
  5. Download and restore. Download the merged .jwlibrary file and restore it in JW Library on your devices.

Frequently asked questions

Do my files get uploaded anywhere?

No. JW Sync runs entirely in your web browser. Your .jwlibrary backup files are never uploaded to any server — every step of the merge happens on your own device, so your notes stay private.

How do I merge JW Library backups from two phones?

Export a backup from JW Library on each device, then open JW Sync, add the first backup as your main file and the others as additional files, and merge. Download the combined .jwlibrary file and restore it in JW Library.

What is a .jwlibrary file?

A .jwlibrary file is the backup that the JW Library app creates. It contains your notes, highlights, bookmarks and tags. JW Sync reads these files and combines them into one.

Does JW Sync work on iPhone and Android?

Yes. JW Sync works in any modern browser on iPhone, iPad, Android and desktop. You can also install it as an app and use it offline.

Will merging overwrite my notes?

No. JW Sync combines your backups, and when the same note was edited on two devices it shows you both versions so you can choose which one to keep before you download.

Is JW Sync free?

Yes. JW Sync is completely free and works without an account or sign-up.

JW Sync processes all data locally — your files never leave your device.
